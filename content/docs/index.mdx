**Acrylic** is a shadcn/ui component registry with a dark frosted-glass (Big Sur /
macOS vibrancy) aesthetic. Color, spacing, and corner radii are lifted from the
official Apple **macOS 26** UI Kit; every component is theme-aware (light / dark)
through one token set.

## Install

Point your project's `components.json` at the Acrylic registry:

```json
{
  "registries": {
    "@acrylic": "https://<this-host>/r/{name}.json"
  }
}
```

Then add components with the CLI:

```bash
npx shadcn add @acrylic/button
```

## Components

Browse the components in the sidebar. Each page has a live **Preview**, the
**Code**, the install command, and a props/variants reference.
