A [Sonner](https://sonner.emilkowal.ski/) toast theme styled as a translucent dark
pill that matches the acrylic aesthetic.

## Installation

```bash
npx shadcn add @acrylic/toaster
```

<ComponentPreview name="toaster-demo" />

## Notes

- Mount `<Toaster />` once near your app root, then call `toast(...)` from `sonner` anywhere.
- Position/offset are sensible defaults you can override via props.
