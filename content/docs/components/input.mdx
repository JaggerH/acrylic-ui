A translucent dark field with a defined hairline border, muted placeholder, and a
macOS-blue focus ring. Compose a leading icon by wrapping the field.

## Installation

```bash
npx shadcn add @acrylic/input
```

## Notes

- Height `h-8`, `rounded-md`, `px-2.5` — macOS control metrics.
- For a leading icon, wrap in a `relative` container, absolutely position the icon, and add `pl-8` to the input (see the search field above).
