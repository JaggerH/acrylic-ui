A NetEase-style "current playlist" panel: a header with a track count and 收藏全部 / 清空
actions over a scrollable list of rows. Each row is a cover (with a hover ▶, persistent on the
playing track), a title (accent-red while playing) + a meta line of quality / MV / VIP badges +
artist, and a trailing duration. Fully presentational and controlled — drop it inside a
[Sheet](/docs/components/sheet) and wire the callbacks to your queue.

## Installation

```bash
npx shadcn add https://acrylic-ui.vercel.app/r/audio-playlist.json
```

## Usage

Click the button to open the playlist in a right-side Sheet.

<ComponentPreview name="audio-playlist-demo" />

```tsx
import { AudioPlaylist } from "@/components/acrylic/audio-playlist"
import { Sheet, SheetContent, SheetTrigger } from "@/components/acrylic/sheet"

<Sheet>
  <SheetTrigger asChild>
    <Button variant="neutral">播放列表</Button>
  </SheetTrigger>
  <SheetContent className="gap-0 p-0">
    <AudioPlaylist
      className="pt-3 [&>div:first-child]:pr-9"
      tracks={tracks}
      currentId={currentId}
      onSelect={play}
    />
  </SheetContent>
</Sheet>
```

## Props

### `AudioPlaylist`

| prop | type | default | description |
| --- | --- | --- | --- |
| `tracks` | `PlaylistTrack[]` | — | the rows to render |
| `currentId` | `string` | — | id of the loaded track (red title + persistent ▶) |
| `heading` | `string` | `播放列表` | panel heading |
| `onSelect` | `(id: string) => void` | — | a row was clicked |
| `onFavoriteAll` | `() => void` | — | 收藏全部 clicked |
| `onClear` | `() => void` | — | 清空 clicked |

Plus all native `<div>` props.

### `PlaylistTrack`

| field | type | description |
| --- | --- | --- |
| `id` | `string` | unique row id |
| `title` | `string` | track title |
| `artist` | `string?` | artist line |
| `cover` | `string?` | cover art URL (glyph fallback) |
| `duration` | `number?` | length in seconds (rendered mm:ss) |
| `vip` | `boolean?` | VIP badge |
| `hires` | `boolean?` | 超清母带 (Hi-Res) badge |
| `mv` | `boolean?` | MV badge |
