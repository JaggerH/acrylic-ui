---
title: Media Box
description: A measured media frame for images, video posters, and player overlays.
---

`MediaBox` is a reusable media frame. It measures the available column width,
keeps a stable image/video preview size, clips children to rounded corners, and
leaves playback or action controls to the caller through `children`.

It does **not** own a video player. Compose an Artplayer, iframe, or custom
controls in the overlay slot when the consuming app needs playback.

## Installation

```bash
npx shadcn add https://acrylic-ui.vercel.app/r/media-box.json
```

## Usage

<ComponentPreview name="media-box-demo" />

```tsx
import { MediaBox } from "@/components/acrylic/media-box"

<MediaBox
  kind="video"
  src={poster}
  naturalWidth={1280}
  naturalHeight={720}
  maxWidth={520}
>
  <div className="absolute inset-0">{/* player overlay */}</div>
</MediaBox>
```

## Props

| prop | type | default | description |
| --- | --- | --- | --- |
| `kind` | `"image" \| "video"` | `"image"` | switches sizing rules |
| `src` | `string` | — | image URL or video poster URL |
| `naturalWidth` / `naturalHeight` | `number` | — | known intrinsic poster/image size |
| `mediaSize` | `{ width: number; height: number } \| null` | — | actual media size, preferred over poster size |
| `maxWidth` | `number` | — | cap the measured frame width |
| `maxHeight` | `number` | image: `510`, video: `507` | cap tall media |
| `frameClassName` | `string` | — | classes for the rounded frame |
| `imageClassName` | `string` | — | classes for the poster/image |
| `fallback` | `ReactNode` | image-off icon | custom broken-image fallback |
| `onSizingChange` | `(snapshot) => void` | — | reports measured sizing details |

Plus native `<div>` props on the wrapper.
