A charcoal translucent pane over a blurred backdrop. Flat at rest (no shadow, no
border); `interactive` adds a hover lift + a soft float shadow. It's a surface
primitive — compose list rows, tiles, and panels on top of it.

## Installation

```bash
npx shadcn add https://acrylic-ui.vercel.app/r/card.json
```

## Example — session list

A real composition: a list of session rows (a 2×2 thumbnail tile with a count
badge, a title / path / date stack, and a trailing trash button that fades in on
hover). `interactive` gives each row the Card's hover lift.

## Props

| prop | type | default | description |
| --- | --- | --- | --- |
| `interactive` | `boolean` | `false` | hover lift + float shadow for clickable cards |

Plus all native `<div>` props.
