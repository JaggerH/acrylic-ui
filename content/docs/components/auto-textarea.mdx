A translucent frosted input that auto-grows with its content up to `maxLines`,
then scrolls with the macOS-style overlay scrollbar.

## Installation

```bash
npx shadcn add @acrylic/auto-textarea
```

## Props

| prop | type | default | description |
| --- | --- | --- | --- |
| `maxLines` | `number` | `3` | grow up to this many rows, then scroll |

Plus all native `<textarea>` props.
