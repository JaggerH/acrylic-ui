A [Radix Dialog](https://www.radix-ui.com/primitives/docs/components/dialog) with a
72px blur + saturate overlay and a translucent white-tint panel. Drop-in API-compatible
with the shadcn Dialog.

## Installation

```bash
npx shadcn add @acrylic/dialog
```

## Notes

- Exports `Dialog`, `DialogTrigger`, `DialogContent`, `DialogClose`, `DialogHeader`, `DialogFooter`, `DialogTitle`, `DialogDescription`.
- The close (X) is the acrylic [Button](/docs/components/button) (`variant="ghost" size="icon"`). Hide it with `<DialogContent showCloseButton={false}>` and dismiss via your own `DialogClose` buttons.
- On open, focus stays on the trigger (the X is not auto-focused, matching macOS). Override with `onOpenAutoFocus` on `DialogContent` if you need to focus a field.
- On a transparent (Tauri/Electron vibrancy) window, see the README for the optional `modal-acrylic` body-paint.
