A charcoal translucent pane over a blurred backdrop. Flat at rest (no shadow, no
border); `interactive` adds a hover lift + a soft float shadow.

## Installation

```bash
npx shadcn add @acrylic/glass-card
```

## Props

| prop | type | default | description |
| --- | --- | --- | --- |
| `interactive` | `boolean` | `false` | hover lift + float shadow for clickable cards |

Plus all native `<div>` props.
